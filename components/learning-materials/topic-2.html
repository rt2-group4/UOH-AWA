HTML elements

HTML elements are divided into:

  • semantic elements
  • non-semantic elements

A semantic element clearly describes its meaning to both the browser and the developer.Some examples of semantic elements are :

  • <form>
  • <table>
  • <article>

Whereas some examples of non-semantic HTML elements are:

  • <div>
  • <span>

HTML <section> Element

The <section> element defines a section in a document. Some examples where a section tag can be used:

  • Chapters
  • Introduction
  • News items
  • Contact information

HTML <article> Element

The <article> element specifies independent, self-contained content. An article should make sense on its own, and it should be possible to distribute it independently from the rest of the web site.

Some examples where you can use an article are:

  • Forum posts
  • Blog posts
  • User comments
  • Product cards
  • Newspaper articles
